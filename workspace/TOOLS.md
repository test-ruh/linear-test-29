# Tools

- `lobster` — runs `workflows/main.yaml`.
- `scripts/data_writer.py` — provisions, writes, and queries result tables in `result-schema.yml`.
- `scripts/github_action.py` — handles branch, commit, and pull request operations when GitHub sync is enabled.
- `openclaw cron` — registered from `cron/jobs.json` by `cron/register-cron.sh`.
